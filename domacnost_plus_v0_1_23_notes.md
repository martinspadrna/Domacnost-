# Domácnost+ v.0.1_23

- Sjednocený fyzický build po v.0.1_22 s cloudovými Domácími úkoly.
- Úkoly umí přidat, načíst z cloudu, odeslat lokální, označit hotovo/vrátit a smazat.
- Cloud používá tabulku household_tasks, která už byla v Supabase vytvořená v předchozím kroku.
- Zachované Balíky z v.0.1_22 a předchozí cloud moduly.
