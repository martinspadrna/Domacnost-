# Domácnost+ v.0.1_20

- Přidaný cloudový základ pro svoz odpadu: `waste_types`, `waste_schedules` a `reminders`.
- Odpad se ukládá podle `household_id`, takže se domácnosti nemíchají.
- Formulář odpadu má opakování, upozornění předem a poznámku.
- Přidané cloud akce: načíst odpad, odeslat lokální svozy, odeslat jednotlivý svoz, smazat svoz.
- Dashboard a karta Domácnost rozlišují lokální/cloud stav odpadu.
- Základ `reminders` je připravený pro pozdější PWA/push notifikace.
